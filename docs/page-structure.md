# 页面结构草图

基于天气数据的旅游出行推荐与可视化系统

> 本文件用于冻结首版展示页面的大致结构和组件摆放，避免前端实现时随意加页或堆无关模块。

---

## 1. 页面总览

首版页面固定为 5 个：

1. 首页兼排行榜页
2. 偏好设置页
3. 城市详情页
4. 城市对比页
5. 历史分析页

所有页面共享的基础组件：

- 顶部导航栏
- 当前数据日期显示
- 数据源状态提示
- 最近更新时间显示
- AQI 模式状态提示
- 空数据提示区
- 错误提示区

---

## 2. 首页兼排行榜页

```text
+----------------------------------------------------------------------------------+
| 顶部导航栏                                                                       |
| Logo / 项目名 | 日期选择 | 手动刷新按钮 | 偏好设置入口 | 历史分析入口 | AQI 状态 |
+----------------------------------------------------------------------------------+
| 页面标题区                                                                       |
| 今日/所选日期旅游推荐 | 说明文案 | 当前城市范围: 10 城市                           |
+-------------------------------------------+--------------------------------------+
| 左侧: 排行榜卡片区                         | 右侧: 当前筛选与偏好摘要             |
| - Top 10 城市列表                          | - 日期                               |
| - 城市名 / 总分 / 排名                     | - 当前偏好                           |
| - 推荐理由摘要                             | - 是否启用 AQI                       |
| - 详情按钮                                 | - 评分规则简述                       |
|                                           | - 最近更新时间                       |
+-------------------------------------------+--------------------------------------+
| 排行榜主图表区                                                                   |
| ECharts 柱状图: 10 城市总分对比                                                   |
+----------------------------------------------------------------------------------+
| 城市速览卡片区                                                                     |
| [城市卡片] [城市卡片] [城市卡片] ...                                             |
| 每张卡片包含: 城市名 / 天气摘要 / 温度 / 雨天标记 / 总分 / 查看详情 / 加入对比      |
+----------------------------------------------------------------------------------+
| 页面底部说明区                                                                     |
| 数据来源说明 | 降级模式说明 | 历史分析入口说明                                     |
+----------------------------------------------------------------------------------+
```

核心组件清单：

- 日期选择器
- 手动刷新按钮
- 偏好摘要卡
- 排行榜列表
- 排行榜柱状图
- 城市速览卡片网格
- AQI 状态提示条
- 最近更新时间提示

## 3. 偏好设置页

```text
+----------------------------------------------------------------------------------+
| 顶部导航栏                                                                       |
+----------------------------------------------------------------------------------+
| 页面标题区                                                                       |
| 个性化偏好设置 | 说明: 修改后会重新计算排行榜和推荐结果                           |
+-------------------------------------------+--------------------------------------+
| 左侧: 偏好表单区                         | 右侧: 影响预览区                     |
| - 降雨敏感度                              | - 当前权重变化说明                   |
| - 温度偏好                                | - 规则变化摘要                       |
| - 风力敏感度                              | - 示例推荐理由变化                   |
| - 旅游风格                                | - AQI 是否启用                       |
| - AQI 敏感度(仅 AQI 可用时显示)           |                                      |
+-------------------------------------------+--------------------------------------+
| 操作区                                                                           |
| 应用偏好 | 恢复默认 | 返回首页兼排行榜页                                          |
+----------------------------------------------------------------------------------+
```

核心组件清单：

- 偏好表单
- 权重变化预览卡
- 示例影响说明卡
- 应用按钮和恢复默认按钮

## 4. 城市详情页

```text
+----------------------------------------------------------------------------------+
| 顶部导航栏                                                                       |
+----------------------------------------------------------------------------------+
| 页面标题区                                                                       |
| 城市名称 | 所选日期 | 当前排名 | 返回首页                                         |
+-------------------------------------------+--------------------------------------+
| 左侧: 基础天气信息卡                     | 右侧: 推荐结论卡                     |
| - 最高温 / 最低温 / 平均温                | - 总分                               |
| - 天气类型                                | - 推荐等级                           |
| - 风力等级                                | - 推荐理由                           |
| - 降雨标记                                | - 偏好影响摘要                       |
| - AQI(若可用)                             |                                      |
+-------------------------------------------+--------------------------------------+
| 分项得分图表区                                                                   |
| 雷达图或柱状图: 温度 / 降雨 / 风力 / 天气类型 / 历史稳定性 / AQI(可选)            |
+----------------------------------------------------------------------------------+
| 趋势图表区                                                                       |
| 未来天气趋势折线图: 近 7 天温度或总分趋势                                         |
+-------------------------------------------+--------------------------------------+
| 左侧: 历史月度统计卡                     | 右侧: 历史分析图                     |
| - 月度平均适宜度                          | - 月度适宜度趋势图                   |
| - 月度雨天频率                            | 或                                   |
| - 舒适天数占比                            | - 雨天频率对比图                     |
+-------------------------------------------+--------------------------------------+
```

核心组件清单：

- 基础天气信息卡
- 推荐结论卡
- 分项得分图
- 未来趋势图
- 历史月度统计卡

## 5. 城市对比页

```text
+----------------------------------------------------------------------------------+
| 顶部导航栏                                                                       |
+----------------------------------------------------------------------------------+
| 页面标题区                                                                       |
| 城市对比 | 日期选择 | 城市 A 选择 | 城市 B 选择                                   |
+----------------------------------------------------------------------------------+
| 对比总览卡片区                                                                   |
| 城市 A 总分 | 城市 B 总分 | 分差 | 推荐结论                                        |
+----------------------------------------------------------------------------------+
| 指标对比表                                                                       |
| 温度 | 降雨 | 风力 | 天气类型 | 历史稳定性 | AQI(可选)                              |
+----------------------------------------------------------------------------------+
| 图表区 1                                                                         |
| 柱状图: 两城市分项得分对比                                                       |
+----------------------------------------------------------------------------------+
| 图表区 2                                                                         |
| 折线图或条形图: 历史月度统计对比                                                 |
+----------------------------------------------------------------------------------+
| 结论区                                                                           |
| 为什么 A 更适合 / 为什么 B 更适合 / 哪些偏好会改变结果                            |
+----------------------------------------------------------------------------------+
```

核心组件清单：

- 双城市选择器
- 对比总览卡
- 指标对比表
- 分项对比图
- 历史月度对比图
- 对比结论卡

## 6. 历史分析页

```text
+----------------------------------------------------------------------------------+
| 顶部导航栏                                                                       |
+----------------------------------------------------------------------------------+
| 页面标题区                                                                       |
| 历史分析 | 城市选择 | 月份选择 | 指标切换                                        |
+-------------------------------------------+--------------------------------------+
| 左侧: 指标说明区                         | 右侧: 当前筛选摘要                   |
| - 指标定义                                | - 当前城市                           |
| - 计算口径                                | - 当前月份/时间范围                  |
| - 数据来源                                | - 当前是否启用 AQI                   |
+-------------------------------------------+--------------------------------------+
| 主图表区                                                                           |
| 月度平均适宜度趋势图 / 月度雨天频率图 / 舒适温度占比图                            |
+----------------------------------------------------------------------------------+
| 排行区                                                                             |
| 历史指标排行榜: 各城市同月适宜度或雨天频率排名                                    |
+----------------------------------------------------------------------------------+
| 结论区                                                                             |
| 历史规律总结 | 适合出游月份提示 | 与当前推荐的关联说明                            |
+----------------------------------------------------------------------------------+
```

核心组件清单：

- 城市选择器
- 月份选择器
- 指标切换器
- 历史分析主图
- 历史排行榜
- 结论说明卡

## 7. 页面跳转关系

```text
[首页兼排行榜页]
   |-- 进入偏好设置 --> [偏好设置页]
   |-- 查看详情 ------> [城市详情页]
   |-- 加入对比 ------> [城市对比页]
   |-- 查看历史分析 --> [历史分析页]

[偏好设置页]
   |-- 应用偏好 ------> [首页兼排行榜页]

[城市详情页]
   |-- 返回首页 ------> [首页兼排行榜页]
   |-- 加入对比 ------> [城市对比页]

[城市对比页]
   |-- 返回首页 ------> [首页兼排行榜页]

[历史分析页]
   |-- 返回首页 ------> [首页兼排行榜页]
```

---

## 8. 首版页面限制

- 不新增欢迎页、登录页、关于页等无关页面
- 不做地图页、景点页、订票页
- 不做复杂分页和多层弹窗流程
- 重点保证首页、详情、对比、历史分析 4 类信息能讲清楚
